Creating a structured document with HTML

Tree-like structure of HTML

DOM = Tree-like structure. This means you can have elements inside elements. Think of a family tree

Indented HTML and boxes

The "wave-like" look of HTML indentations shows the way elements are nested. The more indented means the more nested the element is.

Boxes on boxes on boxes. HTML is all about about boxes. View every page you see as boxes.

Text Editors

Programmers use special text editors for writing code. These editors are designed with coding in mind which makes it easier for coders. Some have really useful features. An example of this in Sublime Text 2 is that you can use the find feature to find a certain tag and then change all of those tags in the doc in one fell swoop. A bit like CSS..
